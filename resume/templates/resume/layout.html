Alibaba Scraping

This is web automation for scraping certain data from Alibaba web site. Python program language, Selenium and Pandas packages are used for making it work.

The program works in a following way – there is an option to add an item name (key word) in the search bar, the new web window is opened and the products related to the word typed in the search bar are searched for.

Each item is opened in the new tab automatically and particular data from the item is scraped. In this project the condition was that the product’s company is established before 2005.

When it reaches 15 companies it is automatically stopped and all the data is saved in an Excel Sheet using Pandas packages.

    • E-commerce Website

    The project created is a web site for e-commerce. Clients can register with their username and password in order to be able to take part in bidding.

    When the profile is created, the client can see which listings are active (Active listings menu), that is, the articles that has not been sold yet. Besides, other menus on the client’s homepage are Log Out, Create, Category and Watchlist menu.

    On the Active listings page, there are listings that are available to be sold. Every item has More info button; when clicking it, detailed info about the product appears. It includes a product description, starting price, current bid (if already bided), who the product is created by and date and time of auction creation. Also, there is an option to add the item to the client’s Watchlist, or to remove it. A client can simply do this by clicking on the Add to/Remove from Watchlist button. Consequently, it will be available to be seen in or removed from the Watchlist menu.

    On the product page, client can make a bid by introducing the price in the Bid value blank space. By clicking on the Make a bid button, the info on the page is updated and it shows the current bid that the client has introduced; the notification Your bid is actual appears. There is also an option to leave a comment about the product bellow the product image.

    Afterwards, if the price introduced suits the seller, the auction can be closed, and in the clients profile appears the notification It has been sold to you.

    When clicking on the Create button, client has an option to create auction. The page Create your product appears and the client should introduce Title, Subtitle, Category, Starting price and Image. That way, the client is the seller of the item.
